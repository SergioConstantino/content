
#### Playbooks
##### New: Account Enrichment - IAM
This playbook aims to retrieve user information from various vendors using the 'iam-get-user'.
Current Supported Integrations:
- Active Directory
- AWS ILM
- Clarizen IAM
- Oracle IAM
- ExceedLMS IAM
- OKTA IAM
- SAP-IAM
- Atlassian IAM
- Envoy IAM
 (Available from Cortex XSOAR 6.5.0).
